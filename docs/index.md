# Test-rack-PC Documentation

## Core components

- CachyOS Linux: A lightweight, rolling-release Linux distribution based on Arch Linux.
- Tasks: A modern task runner for automating development workflows.
- GitHub Actions: A CI/CD platform to automate build, test, and deployment workflows.
- GitLab CI: A continuous integration and delivery (CI/CD) tool built into GitLab
- Containerization: Using Podman and Docker for container management and Kubernetes for orchestration.
